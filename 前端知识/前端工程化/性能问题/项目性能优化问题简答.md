# 一、遇到过首屏加载白屏问题吗，怎么优化？

- 使⽤`CDN`减⼩代码体积，加快请求速度；
- `SSR`通过服务端把所有数据全部渲染完成再返回给客⼾端；
- 路由懒加载，当⽤⼾访问的时候，再加载相应模块；
- 使⽤外链`CSS`，`JS`⽂件；
- 开启`GZIP`压缩；（是⼀种`http`请求优化⽅式，通过减少⽂件体积来提⾼加载速度。
  `html、js、css`⽂件甚⾄`json`数据都可以⽤它压缩，可以减⼩60%以上的体积。（需要后端⽀持））
- 项⽬打包不⽣成`.map`⽂件；
- 使⽤⻣架屏在⾸⻚加载时，显⽰⻚⾯结构轮廓；

# 二、常见的项目优化手段有哪些？

- 減少`HTTP`请求数（避免重复请求）
- 減少`DOM`元素和`DOM`操作
- 使⽤雪碧图
- 压缩合井`JS、CSS`代码
- 避免图⽚`src`为空
- 把样式表放在`link`中
- `css`样式放置在⽂件头部
- `js`脚本放置在⽂件末尾
