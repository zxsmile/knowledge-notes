由一组有序的成员类型构成，表示一个值的类型可以有多种

```
string | number
```

#### 联合类型字面量

由两个或两个以上的成员类型构成，各成员类型之间使用竖线符号`'|'`分隔

```
type NumType  = number | bigint
```

类型成员的类型可以是任意类型。

```
type a = boolean | boolean ===> type a = boolean;
```

它是符合"加法结合率"对部分类型成员使用分组运算符不影响联合类型的结果类型。

```
type t0 = string | number
type t1 = number | string
type t3 = boolean | true |false
type t4 = boolean
```
