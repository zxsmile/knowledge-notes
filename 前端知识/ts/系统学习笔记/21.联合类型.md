# 联合类型

由一组有序的成员类型构成，表示一个值的类型可以有多种

```
string | number
```

## 联合类型字面量

由两个或两个以上的成员类型构成，各成员类型之间使用竖线符号`'|'`分隔

```
type NumType  = number | bigint
```

类型成员的类型可以是任意类型。

```
type a = boolean | boolean ===> type a = boolean;
```

它是符合"加法结合率"对部分类型成员使用分组运算符不影响联合类型的结果类型。

```
type t0 = string | number
type t1 = number | string
type t3 = boolean | true |false
type t4 = boolean
```

假如类型成员中有子类型，那可以将类型成员省略。

```
type t3 = boolean | true |false
type t4 = boolean
```

## 联合类型的类型成员

联合类型的类型成员是由其成员类型决定的

对于联合类型的属性签名，只有共同成员才可以（取交集）

```
interface Circle{
  area: string;
  reduis:number
}

interface Rec;{
  area: string;
  width:number
}

declare const s: Circle | Rec;
s.area = '123'
s.area = 22
s.width //报错
s.reduis //报错
```
