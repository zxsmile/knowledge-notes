# 一、`Object`类型

## `Object`类型的兼容性

除了`undefined`和`null`外，任何值都可以赋值给到`Object`类型。

```

obj={x:0} 
obj ='123' 
obj=11 
obj= true
obj = undefined
```

## `Object`方式-常见错误

`Object`类型用来描述“`Object.prototype`"对象的类型，所以自定义这种属性就不推荐使用`Object`来进行定义了。

```
const point: Object = {x:'xxx', y:123}//不推荐
```
