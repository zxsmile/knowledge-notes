#    命名空间

TS中命名空间是通过namespace关键字来声明，它相当于一种语法糖。

```
namespace Obj.name{
   export namespace Utils{
      function isString(value:any){
         return typeof value === 'string'
      }
   }
}
```

## 多文件中使用命名空间

TS最终会将同名的命名空间合并在一起，即使它不在同一个文件中。

## 文件间的依赖

定义文件间的依赖关系的方法：

#### 使用`tsconfig.json`文件

```
//tsconfig.json
{
   "files":["a.ts","b.ts"]
}
```

#### 使用三斜线指令

```
///<reference path="a.ts" />

namespace App{
  const a = isString('foo')
}
```
