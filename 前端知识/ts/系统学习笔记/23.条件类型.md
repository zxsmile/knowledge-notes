# 条件类型

## 条件类型的定义：

借用`JS`的条件运算符

语法：

```
T extends U ? X : Y
extends:关键字
T、U、X、Y都表示一种类型，条件类型的结果只能为类型X或者类型Y
```

```
type T0 = true extends boolean ? string : number
//string

type T1 = string extends boolean ? string : number
//number
```

## `infer`关键字

```
T extends infer U ? U : Y
```

使用`infer`来定义可推断的类型变量，只允许在条件类型的`true`分支中引用

```
type C1<T> = T extends Array<infer U>? U : never
type T = C1<Array<number>>
```
