# 一、尾端类型

1、是所有其它类型的子类型。

2、也许会出现我们想要一个变量又是数值又是字符串，所以我们就需要一个尾端类型进行定义。

3、也称作0类型或者空类型，它是不包含任何值。

4、尾端类型只有一种:`never`类型

## `never`类型

1、`TS 2.0`引入 使用`never`关键字来定义。

2、`never`类型允许赋值给任何类型;

3、`never`类型是类型结构中的最底层，所以呢，任何类型都不能赋值给`never`类型。

#### 应用场景:

场景一:`never`类型可以作用函数的返回值类型，它表示该函数无法返回任何值。

```
function f():never{
    throw new Error()
}
```

场景二:死循环，永远不可能结束。

```
function infini( ):never{
    while(true){
   	 console.log('12321')
    }
}
```

场景三:使用`never`来完成场景运算;

```
type Exclude<T,U>I = T extends U ? never:T
```

场景四:类型推断，假如编译时，推断没有可用的类型，那么推断结果就是`never`类型。

```
function getLength(mes: string){
    if(typeof mes == 'string'){
       mes;
    else{
       mes; //never
    }
}
```
