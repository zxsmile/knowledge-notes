## 1、`Partial<T>`

能够构造一个新类型，并将实际类型参数T中的所有属性变成可选属性

```
interface A{
  name:string;
  age:number;
}
type T = Partial<A> // name?:string age?:number
const a:T = {}
const b:A = {name:'x',age:12}
```

## 2、`Required<T>`

能够构造一个新类型，并将实际类型参数T中的所有属性变成必选属性

```
interface A{
  name?:string;
  age?:number;
}
type T = Required<A> // name:string age:number
const a:T = {name:'x',age:12}
const b:A = {name:'x'}
```

## 3、`Readonly<T>`

能够构造一个新类型，并将实际类型参数T中的所有属性变成只读属性

```
interface A{
  name:string;
  age:number;
}
type T = Readonly<A> = {name:'x',age:12} //初始化
T.age = 222 //赋值报错
```

## 4、`Record<K , T>`

该工具类型能够使用给定的对象属性名类型和对象属性类型创建一个新的对象类型。

Record<K , T>中的K提供了对象属性名联合类型，类型参数T提供了对象属性的类型。也就是说一个指定的是对象属性名类型，一个是对象属性值的类型。

```
type K= 'x' | 'y'
type T = number
type R = Record<K , T> ===> {x:number,y:number}
const a:R  = {x:1,y:888}
```

## 5、`Pick<K , T>`

从已有对象类型中选取给定的属性及其类型，然后构建出一个新的对象类型。T表示源对象类型，K提供选取的属性名类型，它必须在源对象是已有的

```
interface A{
  x:number;
  y:number;
}
type T0 = Pick<A ,'x'>
type T1 = Pick<A ,'x' | 'y'>
const t:T0 = {x:1}
```

## 6、`Omit<T,K>`

它是和`Pick<K , T>`是互补的，也就是说，它能够从已有对象类型中剔除给定的属性。

```
interface A{
  x:number;
  y:number;
}
type T0 = Omit<A ,'x'>
const t:T0 = {y:1}
type T1 = Omit<A ,'x' | 'y'>
type T1 = Omit<A ,'z'>
```

## 7、`Exclude<T,U>`

该工具类型能够从类型T中剔除所有可以赋值给类型U的类型

```
type T0 = Exclude<'a' | 'b' | 'c', 'a'> // b,c
type T1 = Exclude<string | (() => void), Function> // string
type T2 = Exclude<string | number , string> // number
const t2:T2 = 125
```

## 8、`Extract<T,U>`

`Extract`是和 `Exclude` 是互补的，它是能够从类型`T`中获取所有可以赋值给类型`U`的类型

```
type T0 = Extract<'a' | 'b' | 'c', 'a'> // a
type T1 = Extract<string | (() => void), Function> // Function
type T2 = Extract<string | number | boolean , boolean> // boolean
const t2:T2 = true
type T3 = Extract<string | number | boolean , boolean | string> // boolean string
const t3:T3 = true
const t3:T3 = '55666'
```

## 9、`NonNullable`

该工具类型能够从类型T中剔除`Null`类型和`undefined`类型，构造出一个新类型

```
type T2 = NonNullable<string | number | undefined 
null>  // string | number
const t1:T0 = 122
```
