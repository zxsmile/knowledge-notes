# 一、构造函数类型字面量

构造函数类型字面量是定义构造函数类型的方法之一，它能够指定函数参数类型、函数的返回值类型以及泛型类型参数

```
new (ParameterList) => Type
```

```
let a = new Error()
let b = new Error('Error message')
let ErrorConstructor:new (message?:string) => Error
```

## 构造签名

```
{

   new (ParameterList)：Type

}
```

```
let Dog:{new (name:string):object}
Dog = class{
  private name:string;
  constructor(name:string){
     this.name = name
  }
}
let dog = new Dog('keke')

```

当构造函数类型字面量是仅包含一个构造签名的对象类型字面量的简写形式。

```
let li:{new (name:stirng):object}
相当于
let li:new (name:stirng)=>object
```

## 调用签名和构造签名

```
declare const F:{
  new (x:number):Number,
  (x:number):number
}

const a:number = F(1)//普通函数调用
const b:Number = new F(1) //构造函数调用
```
