# 一、`enum`类型

`enum`:是计算机语言中的一种数据类型。枚举类型，
在`TS`中，`enum`是一种原始类型，它通过关键字`enum`来定义的。分类:根据枚举类型中的成员进行的

## 数值型枚举

1、`number`类型的子类型。它由一组命名的数值常量构成。

2、我们可以使用访问对象属性一样去访问枚举成员。

```
enum Direction{
    Up,// 0
    Down,// 1
    Left, // 2
    Right // 3
}
const Dir: Direction = Direction.Up;
console.log(Direction.Up,Direction.Up,Direction.Left, Direction.Right)
//0 1 2 3
```

3、每个数值型枚举成员都表示一个具体的数字，如果在定义时没有设置，TS会自动计算枚举成功的值，会给第一项设置为0，后面的，则是等于前一个枚举成员的值加1;可以在定义时，为一个或者多个枚举成员设置初始值。

```
enum Direction{
    Up,        // 0
    Down = 20, // 20
    Left,      // 21
    Right      // 22
}
const Dir: Direction = Direction.Up;
console.log(Direction.Up,Direction.Up,Direction.Left, Direction.Right)
//0 20 21 22
```

4、因为枚举成员都是`number`类型，所以我们可以直接设置为`number`

```
enum Direction{
    Up,        // 0
    Down = 20, // 20
    Left,      // 21
    Right      // 22
}
const Dir: number = Direction.Up;
console.log(Direction.Up,Direction.Up,Direction.Left, Direction.Right)
//0 20 21 22
```

5、`number`类型可以直接赋值给数值型枚举类型，必须在枚举列表中。

```
enum Direction{
    Up,        // 0
    Down = 20, // 20
    Left,      // 21
    Right      // 22
}
const Dir: Direction = 30;//报错
const Dir: Direction = 20;//正确
```

## 字符串枚举

1、枚举成员的值是字符串。

2、必须初始化:字符串字面量、或者使用其它枚举成员。

**注意:没有自增长的行为。**

3、允许将字符串枚举类型赋值给string类型。

**注意:但是反过来，它是不允许将string类型直接赋值给字符串枚举类型，即使你的枚举
列表中包含这个值也不可以。这个是和数值型不同的点。**

```
enun StrDic{
//字面量
Up = 'up',
Down = 'down',
Left = 'left',
Right = 'right',
//其它枚举成员
U = Up

const str:StrDic = StrDic.Up

const str:string = StrDic.Up
const str:StrDic = 'up' //报错

```

## 异构型枚举:

TS允许在一个枚举中同时定义数值型和字符串型枚举成员。

**注意:在数值型没有初始值时，必须放在字符串枚举成员前面去写**

```
enum Color {
    White = 'white',
    Black =10
}
enum Color {
	Black,
    White = 'white', 
}

console.log(Color.Black)
```

## 枚举成员映射

- 不论是哪种类型的枚举，都可以通过成员名去访问枚举成员值。

- 所以类型枚举都是可以通过枚举成员名去访问枚举成员值。

  **但是只有数值型枚举还可以通过枚举成员值去访问到枚举成员的名。**

```
enum Bool{
   A=1，
   B=2
}
enum Str{
   C= 'xx',
   D = 'xxx'
}
console.log(Bool.A, Bool.B) // 1 2
console.ldg(Str.C,Str.D)// 'xx' 'xxx'
console.log(Bool[Bool.A]，Bool[2]) // A B
console.log(Str[Str.C]，Str['xxx']) //报错
```

## 常量枚举成员

每个枚举成员都有一个值，我们就根据成员的值定义可以将枚举成员分成常量枚举成员:初始值是

常量枚举表达式，那么该成员是常量枚举成员。

若枚举成员没有初始值，`ts`会自动给个0;

若枚举没有初始值，且上一个成员是数值的话，那该成员会是上一个成员加1，但是假如不是数值的话，就会报错。

```
enum Foc{

    A ='a',

    B = A,

    C = 'C',

    D = -1,

    E = 1 + 2,

    F = (1+2) * 3

}
```

## 计算枚举成员

除常量枚举成员，其它的都属于计算枚举成员:

```
enum Bo{
A = 'A'.length,
B = Math.abs(1.23)

}
```

## `const`枚举类型


作用:是因为`ts`编译枚举类型时呢，在`JS`中既生成了正向隐射，又生成反向映射，但是对于我们来说，我们只需要正向，所以这个就可以用`cosnt`枚举类型来实现

编译后的内容:除了正向映射关系，还额外添加了注释，注释内容则是枚举成员的名字。

方法: `const enum`关键字

```
const enum NDir{
    up,
    Down,
    Left,
    Right
}
const E:NDir = NDir.Down
```



```
//编译后
var E = 1 /* NDir.Down */;
```
