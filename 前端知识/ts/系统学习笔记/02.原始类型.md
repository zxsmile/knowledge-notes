# 一、原始类型

`JS`中的所有原始类型都有与之对应的`ts`类型。

#### 1、`string`:===> `string`来表示

```
let str:string = '123'
```

#### 2、`boolean`: ===> `boolean`来表示

```
let yes:boolean= true
```

#### 3、`number`: ===> `number`来表示

```
let num:number= 222
```

#### 4、`null`:  ===>`null`来表示

```
let obj:null= null
```

#### 5、`undefined`: ===>`undefined`来表示

```
let foo:undefined= undefined
```

#### 6、`BigInt`: ===>`bigint`来表示

该类型能够表示任意精度和整数，但也仅能表示整数

```
let bin:bigint= 10n
```

#### 7、`Symbol`: ===>`symbol`来表示

```
let key:symbol= Symbol()
```

对于Symbol类型是没有字面量:也就没有一个固定值。所以ts呢有了一个其它类型

#### 8、`unique symbol` ===> `unique symbol`

1、主要用途是用作接口、类等类型的定义。

2、将一个`Symbol`值与声明它的标识符绑定在一起。并通绑定了该`Symbol`值的标识符来表示`Symbol`字面量。

3、它的绑定关系不可变的，所以必须使用`const`来定义或者使用`readonly`来修饰。

```
const s0: unique symbol =  Symbol()

interface WithUS{

   readonly b : unique symbol;

}
```
