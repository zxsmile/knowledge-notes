### 二、变量 ###

- ECMAScript的变量松散类型的，所谓松散类型就是可以用来保存任何类型的数据。

- 换句话说，每个变量仅仅是一个用于保存值的占位符而已。

- 使用var操作符定义的变量将成为定义该变量的作用域中的局部变量。也就是说如果在函数中使用var定义一个变量，那么这个变量在函数退出后就会被销毁。

  ```
  function test(){
    var a = 'hi' //局部变量
  }
  
  test()
  alert(a); //错误
  ```

- 省略var操作符，变量就成了全局变量。这样，只要调用过一次函数，这个变量就有了定义，就可以在函数外部的任何地方被访问到、

  ```
  function test(){
    a = 'hi' //全局变量
  }
  
  test()
  alert(a); //hi
  ```
